🛠️ Detalles Técnicos: Estructura de Granos

Metadata de Tortilla: Uso de string tipoTortilla para diferenciar platos basados en trigo (influencia de las bananeras) y maíz (herencia ancestral).

Votación Dual: Sistema de Likes/Dislikes para medir la popularidad de las recetas hondureñas.

Optimización en Base: Estructura de datos diseñada para minimizar el uso de storage y reducir el gas.
