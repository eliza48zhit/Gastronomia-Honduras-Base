🛠️ Detalles Técnicos: Estructura de Granos e Identidad

Este contrato evoluciona la arquitectura de la serie al añadir:

Metadata de Tortilla (string): Uso de la variable tipoTortilla para diferenciar platos basados en trigo (influencia de la costa norte) y maíz (herencia del interior), un factor determinante en la identidad hondureña.

Sistema de Reputación Dual: Funciones inmutables para darLike y darDislike que permiten medir la aceptación popular de las recetas registradas.

Optimización de Almacenamiento: Estructura diseñada para minimizar el uso de storage y reducir el costo de gas en cada interacción dentro de la red Base.
